# Just Pong 🏓

A fast-paced, neon-styled remake of the absolute arcade classic, built completely from scratch using HTML5 Canvas and Vanilla JavaScript. No external libraries or installations required!

## 🕹️ How to Play
1. Click the **Live Demo** link above.
2. Move your computer mouse (or slide your finger on a laptop trackpad) **up and down**.
3. Your paddle (the blue one on the left) will follow your movement automatically.
4. Defeat the computer (the magenta paddle on the right) by being the first to score **3 points**!

## 💻 Behind the Code (For Developers)
If you are interested in the technical side, this project showcases:
* **Smooth Animation Loops:** Powered by `requestAnimationFrame` for consistent 60fps performance.
* **Vector-Based Physics:** Real-time calculation of angles and ball acceleration on every paddle hit.
* **Basic AI Engine:** A responsive automated opponent with smooth position interpolation (`lerp`).
